## PL/0 编译器（始终并行）

PL/0 编译器是上机检查的核心交付物，贯穿整个课程设计期间。

### 代码文件

| 文件 | 路径 | 说明 |
|------|------|------|
| PL.cpp | `C/src/PL/PL.cpp` | 递归下降编译器主程序（符号表 + 错误处理 + 语法制导翻译） |
| common.h | `C/src/PL/common.h` | 符号表定义、CList 适配 |
| interpret.cpp | `C/src/interpret/interpret.cpp` | 抽象机解释执行引擎 |
| common.h | `C/src/interpret/common.h` | 共用定义（栈大小等） |
| conio.h | `C/include/conio.h` | Linux 替代 DOS conio.h 的 termios shim |
| clist.h | `C/include/clist.h` | 纯 C++ CList shim（替代 MFC CList） |
| PL.exe | `C/bin/PL.exe` | 编译产物：PL/0 编译器 |
| INTERPRET.exe | `C/bin/INTERPRET.exe` | 编译产物：解释器 |
