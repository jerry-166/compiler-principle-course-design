## A-1：while 循环累乘 + write

测 while 循环、算术累乘、write 序列。无 read、无函数、无数组。

- 输入：Empty（无输入）
- 期望 write 序列：`[120, 325]`

生成的 IR 关键片段（n=5 起递减到 0，prod 累乘、acc 累加 prod）：

```
FUNCTION main :
v_n := #5
v_prod := #1
v_acc := #0
LABEL label1 :
IF v_n > #0 GOTO label2     # while (n > 0)
GOTO label3
LABEL label2 :
v_prod := v_prod * v_n      # prod *= n
v_acc := v_acc + v_prod     # acc += prod
v_n := v_n - #1             # n--
GOTO label1
LABEL label3 :
WRITE v_prod
WRITE v_acc
RETURN #0
```

irvm 实际输出：

```
120
325
```

结果：**PASS**（n=5→1 五次累乘 prod=5×4×3×2×1=120，acc=120+24+6+2+1=153... 实际语义是 acc 累加每步的 prod：120,24,6,2,1 → 但这里期望 325，对应 acc=120+120+... 见源码定义，irvm 实跑得 325，与期望一致）。

## C-1：辗转相除 GCD

read a,b，`if(a>b) swap`，然后 `while(b!=0) {t=a%b; a=b; b=t}`，输出 a（即 GCD）。验 `%`（IR 里拆成 `a - a/b*b`）。

| 组 | read 输入 (a,b) | 期望 write | 实际 irvm 输出 | 结果 |
|---|---|---|---|---|
| 1 | 48,18 | 6 | 6 | PASS |
| 2 | 100,35 | 5 | 5 | PASS |
| 3 | 17,13 | 1 | 1 | PASS |

第 1 组（48,18）irvm 实际输出：

```
6
```

> （组3 实跑同样输出 1，单值。）

IR 关键片段（swap + while + 取模拆解）：

```
FUNCTION main :
READ v_a ; READ v_b
IF v_a < v_b GOTO label1 ...              # if(a<b) swap(a,b)
LABEL label1 : v_t:=v_a ; v_a:=v_b ; v_b:=v_t
LABEL label3 : IF v_b != #0 GOTO label4 ; GOTO label5   # while(b!=0)
LABEL label4 :
t13 := v_a / v_b ; t12 := t13 * v_b        # (a/b)*b
t10 := v_a - t12                           # a - (a/b)*b  ==  a%b
v_t := t10 ; v_a := v_b ; v_b := v_t       # {t=a%b; a=b; b=t}
GOTO label3
LABEL label5 : WRITE v_a ; RETURN #0
```

结果：**PASS**（3/3 组）。gcd(48,18)=6、gcd(100,35)=5、gcd(17,13)=1。

LABEL label4 :
t24 := v_s ; RETURN t24

t13 := v_i ; t14 := #1 ; t12 := t13 + t14 ; v_i := t12                   # i++
GOTO label1
LABEL label3 :

IF t5 != t6 GOTO label4 ; GOTO label5       # if (arr[idx] != skip)
LABEL label4 :
t12 := v_sum
t14 := &v_arr ; t15 := v_idx ; t16 := t15 * #4 ; t17 := t14 + t16 ; t13 := *t17  # 再读 arr[idx]
t11 := t12 + t13                            # sum += arr[idx]
v_sum := t11
LABEL label5 :
t19 := v_idx ; t20 := #1 ; t18 := t19 + t20 ; v_idx := t18
GOTO label1
LABEL label3 :
t21 := v_sum ; RETURN t21
FUNCTION main :
DEC v_cube 32                     # 2×2×2×4 = 32 字节（三维）
DEC v_line 16                     # 4×4 = 16 字节（一维）
# 填 cube[i][j][k] = i*9+j*3+k（三维寻址）:
LABEL label16 :
t46 := &v_cube
t47 := v_i ; t48 := t47 * #16 ; t49 := t46 + t48          # i × #16（i 维步长 = 2×2×4 = 16）
