## A-1：while 循环累乘 + write

测 while 循环、算术累乘、write 序列。无 read、无函数、无数组。

- 输入：Empty（无输入）
- 期望 write 序列：`[120, 325]`

生成的 IR 关键片段（n=5 起递减到 0，prod 累乘、acc 累加 prod）：

```
FUNCTION main :
v_n := #5
v_prod := #1
v_acc := #0
LABEL label1 :
IF v_n > #0 GOTO label2     # while (n > 0)
GOTO label3
LABEL label2 :
v_prod := v_prod * v_n      # prod *= n
v_acc := v_acc + v_prod     # acc += prod
v_n := v_n - #1             # n--
GOTO label1
LABEL label3 :
WRITE v_prod
WRITE v_acc
RETURN #0
```

irvm 实际输出：

```
120
325
```

结果：**PASS**（n=5→1 五次累乘 prod=5×4×3×2×1=120，acc=120+24+6+2+1=153... 实际语义是 acc 累加每步的 prod：120,24,6,2,1 → 但这里期望 325，对应 acc=120+120+... 见源码定义，irvm 实跑得 325，与期望一致）。

## A-2：多条件 read + 嵌套 if + 多 write

读三个数 a,b,c，按 a>b / b>c / a>c 三个条件累加 t（+1/+2/+4），再按 a+b+c>100 输出 1000/2000，按 a*b/2>c 输出 3000/4000。

| 组 | read 输入 (a,b,c) | 期望 write | 实际 irvm 输出 | 结果 |
|---|---|---|---|---|
| 1 | 40,30,20 | 7,2000,3000 | 7,2000,3000 | PASS |
| 2 | 10,50,8 | 6,2000,3000 | 6,2000,3000 | PASS |
| 3 | 60,50,5 | 7,1000,3000 | 7,1000,3000 | PASS |

第 1 组 irvm 实际输出（其余两组同理逐行）：

```
7
2000
3000
```

IR 关键片段（条件翻译 + 算术 + 三段独立 if）：

```
FUNCTION main :
READ t1 ; v_a := t1
READ t2 ; v_b := t2
READ t3 ; v_c := t3
v_t := #0
IF v_a > v_b GOTO label1 ; GOTO label2     # if(a>b) t+=1
LABEL label1 : v_t := v_t + #1 ; LABEL label2 :
IF v_b > v_c GOTO label3 ; ...             # if(b>c) t+=2
IF v_a > v_c GOTO label5 ; ...             # if(a>c) t+=4
WRITE v_t
IF (v_a+v_b+v_c) > #100 GOTO label7        # a+b+c>100 ? 1000 : 2000
  WRITE #1000 ; GOTO label9 ; LABEL label8 : WRITE #2000
IF (v_a*v_b/#2) > v_c GOTO label10         # a*b/2>c ? 3000 : 4000
  WRITE #3000 ; ... LABEL label11 : WRITE #4000
RETURN #0
```

结果：**PASS**（3/3 组）。

## A-3：循环填一维数组 + read 下标取值

循环 `for i in 0..5` 把 `buf[i] = i*i+3`，再 read k，输出 `buf[0], buf[k], buf[5]`。

| 组 | read 输入 (k) | 期望 write | 实际 irvm 输出 | 结果 |
|---|---|---|---|---|
| 1 | 2 | 3,7,28 | 3,7,28 | PASS |
| 2 | 4 | 3,19,28 | 3,19,28 | PASS |
| 3 | 0 | 3,3,28 | 3,3,28 | PASS |

第 1 组（k=2）irvm 实际输出：

```
3
7
28
```

IR 关键片段（一维数组寻址：`&v_buf + i*#4`，`DEC v_buf 24` 开 6×4=24 字节）：

```
FUNCTION main :
DEC v_buf 24
v_i := #0
LABEL label1 :                       # for(i=0; i<6; i++)
IF v_i < #6 GOTO label2 ; GOTO label3
LABEL label2 :
t4 := &v_buf ; t6 := v_i * #4 ; t7 := t4 + t6     # &buf[i]
t9 := v_i * v_i ; t8 := t9 + #3                   # i*i+3
*t7 := t8                                          # buf[i] = i*i+3
v_i := v_i + #1 ; GOTO label1
LABEL label3 :
READ t16 ; v_k := t16
t18 := &v_buf ; t21 := t18 + #0*#4 ; WRITE *t21    # buf[0]=0*0+3=3
t23 := &v_buf ; t26 := t23 + v_k*#4 ; WRITE *t26   # buf[k]
t28 := &v_buf ; t31 := t28 + #5*#4 ; WRITE *t31   # buf[5]=5*5+3=28
RETURN #0
```

结果：**PASS**（3/3 组）。buf[0]=3、buf[2]=2×2+3=7、buf[4]=4×4+3=19、buf[5]=28，全部正确。

## A-4：双重循环 + 奇数判定 + 求和

`for i=1..n, for j=1..n, if (i 是奇数) sum += i*j`。验嵌套 for、`i/2*2 != i` 判奇。

| 组 | read 输入 (n) | 期望 write | 实际 irvm 输出 | 结果 |
|---|---|---|---|---|
| 1 | 3 | 24 | 24 | PASS |
| 2 | 2 | 3 | 3 | PASS |

第 1 组（n=3）irvm 实际输出：

```
24
```

IR 关键片段（嵌套 LABEL/GOTO + 除法判奇）：

```
FUNCTION main :
READ v_n ; v_sum := #0 ; v_i := #1
LABEL label1 : IF v_i <= v_n GOTO label2 ; GOTO label3   # for i
LABEL label2 : v_j := #1
LABEL label4 : IF v_j <= v_n GOTO label5 ; GOTO label6   # for j
LABEL label5 :
t11 := v_i / #2 ; t9 := t11 * #2                         # i/2*2
IF t9 != v_i GOTO label7 ; GOTO label8                   # i 是奇数?
LABEL label7 : v_sum := v_sum + v_i * v_j                # sum += i*j
LABEL label8 : v_j := v_j + #1 ; GOTO label4
LABEL label6 : v_i := v_i + #1 ; GOTO label1
LABEL label3 : WRITE v_sum ; RETURN #0
```

结果：**PASS**（2/2 组）。n=3 时奇数 i=1,3：i=1 贡献 1×(1+2+3)=6，i=3 贡献 3×(1+2+3)=18，合计 24。

| 1 | 6 | 8 | 8 | PASS |


IR 关键片段（PARAM/ARG/CALL 递归）：

```
FUNCTION fib :

IF v_n <= #1 GOTO label1 ; GOTO label2     # if (n<=1) return n
LABEL label1 : RETURN v_n
LABEL label2 :
t7 := v_n - #1 ; ARG t7 ; t5 := CALL fib   # fib(n-1)
t10 := v_n - #2 ; ARG t10 ; t6 := CALL fib # fib(n-2)
RETURN t5 + t6
FUNCTION main :

```

结果：**PASS**（3/3 组）。fib(6)=8、fib(5)=5、fib(0)=0。

## B-1：数组写入 + 找最大 + 调函数

|---|---|---|---|---|
| 1 | 3,8,2,8,1,0 | 1,8,17 | 1,8,17 | PASS |
| 2 | 5,4,3,9,9,2 | 3,9,19 | 3,9,19 | PASS |
| 3 | 1,2,3,4,5,0 | 4,5,11 | 4,5,11 | PASS |


IR 关键片段（数组 READ 写入 + 函数 CALL 取返回值）：

```
FUNCTION scoreAt : PARAM v_val ; RETURN v_val * #2 + #1
FUNCTION main :
DEC v_vals 24
v_i := #0
LABEL label1 : IF v_i < #6 GOTO label2 ; GOTO label3   # 读 6 个数进 vals
LABEL label2 : t9 := &v_vals + v_i*#4 ; READ t13 ; *t9 := t13
              v_i := v_i + #1 ; GOTO label1
LABEL label3 :
v_best := *(&v_vals + 0) ; v_bestIdx := #0
LABEL label4 : IF v_i < #6 GOTO label5 ; GOTO label6  # 扫描找最大
LABEL label5 : IF *(&v_vals + v_i*#4) > v_best GOTO label7 ...
LABEL label7 : v_best := *(&v_vals + v_i*#4) ; v_bestIdx := v_i
LABEL label6 :
WRITE v_bestIdx ; WRITE v_best
ARG v_best ; t43 := CALL scoreAt ; WRITE t43
RETURN #0
```

结果：**PASS**（3/3 组）。组1：最大值 8 在下标 1，score=8×2+1=17。

```
FUNCTION clampAdd :
```

## C-1：辗转相除 GCD

read a,b，`if(a>b) swap`，然后 `while(b!=0) {t=a%b; a=b; b=t}`，输出 a（即 GCD）。验 `%`（IR 里拆成 `a - a/b*b`）。

| 组 | read 输入 (a,b) | 期望 write | 实际 irvm 输出 | 结果 |
|---|---|---|---|---|
| 1 | 48,18 | 6 | 6 | PASS |
| 2 | 100,35 | 5 | 5 | PASS |
| 3 | 17,13 | 1 | 1 | PASS |

第 1 组（48,18）irvm 实际输出：

```
6
```

> （组3 实跑同样输出 1，单值。）

IR 关键片段（swap + while + 取模拆解）：

```
FUNCTION main :
READ v_a ; READ v_b
IF v_a < v_b GOTO label1 ...              # if(a<b) swap(a,b)
LABEL label1 : v_t:=v_a ; v_a:=v_b ; v_b:=v_t
LABEL label3 : IF v_b != #0 GOTO label4 ; GOTO label5   # while(b!=0)
LABEL label4 :
t13 := v_a / v_b ; t12 := t13 * v_b        # (a/b)*b
t10 := v_a - t12                           # a - (a/b)*b  ==  a%b
v_t := t10 ; v_a := v_b ; v_b := v_t       # {t=a%b; a=b; b=t}
GOTO label3
LABEL label5 : WRITE v_a ; RETURN #0
```

结果：**PASS**（3/3 组）。gcd(48,18)=6、gcd(100,35)=5、gcd(17,13)=1。

## C-2：数位和 digitSum（含负数取绝对值）

| 组 | read 输入 (v) | 期望 write | 实际 irvm 输出 | 结果 |
|---|---|---|---|---|
| 1 | 12345 | 15 | 15 | PASS |
| 2 | 909 | 18 | 18 | PASS |
| 3 | 7 | 7 | 7 | PASS |
| 4 | 0 | 0 | 0 | PASS |

第 1 组（12345）irvm 实际输出：

```
15
```

IR 关键片段（负数取反 + 数位拆解循环）：

```
FUNCTION digitSum :
PARAM v_n ; v_s := #0 ; v_x := v_n
IF v_x < #0 GOTO label1 ...                # if(x<0) x=-x
LABEL label1 : v_x := #0 - v_x
LABEL label3 : IF v_x > #0 GOTO label4 ; GOTO label5   # while(x>0)
LABEL label4 :
t15 := v_x / #10 ; t14 := t15 * #10
t12 := v_x - t14                           # x%10
v_s := v_s + t12                           # s += x%10
v_x := v_x / #10                           # x /= 10
GOTO label3
LABEL label5 : RETURN v_s
FUNCTION main :
READ v_v ; ARG v_v ; t24 := CALL digitSum ; WRITE t24 ; RETURN #0
```

结果：**PASS**（4/4 组）。1+2+3+4+5=15、9+0+9=18、7、0。
